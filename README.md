# cybersecurity-labs

Laboratórios práticos desenvolvidos para estudar Vulnerability Assessment, Network Security e System Hardening.


## Objetivos

- Construção de ambientes vulneráveis
- Enumeração de serviços
- Avaliação de vulnerabilidades
- Correção de falhas
- Reavaliação do ambiente

## Tecnologias

- Kali Linux
- Ubuntu Server
- Nmap
- Samba
- Apache
- MariaDB
